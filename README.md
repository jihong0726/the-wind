每一章交稿前硬过以下四关：

| 项 | 阈值 | 验证方式 |
|---|---|---|
| 字数 | ≥ 3000 汉字 | `re.findall(r'[\u4e00-\u9fff]', text)` |
| AI 指纹总率 | ≤ 5‰ | 11 类 T1/T2 模式正则扫描 |
| 单类 AI 指纹 | ≤ 2 处/章 | 同上 |
| Seba 手法注入 | ≥ 6 处 / ≥ 4 种 | 问句自答 / 自嘲转折 / 物件拟人 / 结构反差 / 环境映射 / 长短句 / 冷峻留白 / 具体动作结尾 |

**禁用词** 呵 / 冷笑 / 勾起嘴角 / 不由得 / 不禁 / 他感到 / 她感到 / 心中一动 / 脑海中 …（详见 `00_风格守护机制.md`）
