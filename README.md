## ✍️ 叙事与写作规范 (Style Guide)

### 1. 视角交替 (POV Rules)
* **单数章 = 嘉弘 (Jiahong)**
    * 时间格式：中文 (e.g., "下午三点", "凌晨两点")
    * 开头必现锚点：键盘 / 外卖 / 系统提示音 / 潮湿的墙角
* **双数章 = 林冬 (Winter)**
    * 时间格式：数字 (e.g., "15:00", "02:00")
    * 开头必现锚点：妆发 / 镜头 / 通告车 / 耳返

### 2. 禁令 (Restrictions)
* **树洞描写**：禁止上帝视角。只描写屏幕可见文字（正在输入... / 对方已断开）。
* **情绪克制**：不走金句堆叠路线，情绪要通过动作（如推眼镜、揉眉心）来体现。

### 3. 章节衔接规则 (Chapter Link Rule)

- 每章结尾必须落在一个可见动作或屏幕状态上  
  - 例：已断开 关闭工单 戴上耳返 拉上车门 按下电梯键
- 下一章开头前 2–3 句必须接住上一章的落点  
  - 用同一动作的延续 或 反向动作开场  
  - 例：上一章结尾已断开 下一章开头重新打开树洞并开始匹配
- 钩子只能来自当前视角可见内容  
  - 树洞只写屏幕可见状态  
  - 嘉弘线优先用桌面 流程 提示音 工单  
  - 林冬线优先用通告 车内 灯光 妆发 耳返
- 章节结尾不做总结 不落地结论  
  - 只留下一步动作或悬而未决的状态
