# 📑 章节总大纲 (Chapter Index & Roadmap)

> **当前版本**: v2.0 (T国提速版)
> **规划范围**: 第一卷全 (Ch 1 - Ch 54)
> **核心节奏**: 压抑 (1-15) -> 逃离 (16-28) -> 交汇 (29-40) -> 默契 (41-54)

## 🌧️ 第一阶段：平行线与压抑 (The Parallel Lines)
*关键词：无窗房、通告车、噪音、渴望呼吸*

| 章号 | 视角 | 核心事件 | 关键钩子/备注 |
| :--- | :--- | :--- | :--- |
| **Ch 01** | 嘉弘 | **夜班上线**。描写无窗房的压抑与机械键盘声。处理第一单纠纷。 | 钩子：外卖袋挂在门把上。 |
| **Ch 02** | Winter | **通告间隙**。保姆车内的幽闭感。团内关系（Karina递水）。 | 钩子：透过车窗看雨。 |
| **Ch 03** | 嘉弘 | **树洞初现**。下班后的孤独，第一次点开 App 但只停在首页。 | 状态：点下“聆听”又取消。 |
| **Ch 04** | Winter | **深夜练习**。高强度练舞后的虚脱。第一次尝试“倾诉”但秒关。 | 状态：输入框空白。 |
| **Ch 05** | 嘉弘 | **陌生匹配**。第一次正式连线，遇到路人，体验“2+1”断开机制。 | 规则展示章，不带感情。 |
| **Ch 06** | Winter | **偷拍惊魂**。在便利店附近发现镜头，恐惧爆发。躲回车里开树洞。 | 情绪：惊弓之鸟。 |
| **Ch 07** | 嘉弘 | **工作话术**。处理大额退款，展示职业素养。树洞里依然沉默。 | 对比：工作多话 vs 树洞寡言。 |
| **Ch 08** | Winter | **舆论高墙**。刷到 CP 粉视频，感受到被凝视的压力。扣手机继续练舞。 | 动作：狠摔手机进沙发。 |
| **Ch 09** | 嘉弘 | **情绪承接**。匹配到一个情绪崩溃的路人，他用客服技巧安抚。 | 伏笔：他是个好的倾听者。 |
| **Ch 10** | Winter | **卸妆时刻**。车内卸下一半妆容，像卸下盔甲。树洞里发了一句“累”。 | 状态：对方无反应 -> 断开。 |
| **Ch 11** | 嘉弘 | **噪音侵入**。室友“麦克风”直播喊麦穿透墙壁。他戴上降噪耳机也无用。 | **推手 1**：对环境忍无可忍。 |
| **Ch 12** | Winter | **宿舍夜话**。回到宿舍，队友的热闹衬托她的孤独。她躲进被子看树洞。 | 心理：人群中的孤独。 |
| **Ch 13** | 嘉弘 | **主动倾诉**。为了理解“倾诉者”的心态而尝试，结果觉得矫情删了。 | 尝试失败。 |
| **Ch 14** | Winter | **好奇聆听**。第一次点聆听，遇到普信男，果断退出。 | 确立她的高冷/防御机制。 |
| **Ch 15** | 嘉弘 | **深夜买水**。走廊里的陌生脚步声。回到电脑前，对树洞有了期待。 | 氛围：他在等一个特定的人。 |

## ✈️ 第二阶段：危机与逃离 (The Escape)
*关键词：外派机会、T国行程、必须离开的理由*

| 章号 | 视角 | 核心事件 | 关键钩子/备注 |
| :--- | :--- | :--- | :--- |
| **Ch 16** | Winter | **灯光过载**。棚拍闪光灯引发轻微 PTSD。经纪人提及“去 T 国暂避风头”。 | **推手 2**：被动流放的开始。 |
| **Ch 17** | 嘉弘 | **星河璀璨**。第一次用固定开场白匹配到她。 | **Key Moment**：初次连接。 |
| **Ch 18** | Winter | **识别建立**。看到那句开场白，回了句“又是你”。没有断开。 | 关系：0% -> 5%。 |
| **Ch 19** | 嘉弘 | **克制对话**。不问隐私，只聊“今天吃了什么”。 | 风格：安稳、不越界。 |
| **Ch 20** | Winter | **时间标记**。用“通告结束”代替具体时间。树洞成为她的安全屋。 | 信任建立。 |
| **Ch 21** | 嘉弘 | **室友爆发**。麦克风通宵直播。嘉弘彻夜未眠，盯着富胖达“外派 T 国”的邮件。 | 决策点：必须走了。 |
| **Ch 22** | Winter | **行程敲定**。公司正式定下 T 国行程。她看着护照，有些期待逃离。 | 决策点：接受安排。 |
| **Ch 23** | 嘉弘 | **申请提交**。点击“申请转岗：T国大额专线”。理由写了：夜班津贴。 | 动作：点击发送。 |
| **Ch 24** | Winter | **临行告别**。最后一次在练习室，把柜子清空。 | 仪式感：告别过去。 |
| **Ch 25** | 嘉弘 | **最后的一晚**。打包行李。键盘膜没撕，直接塞进箱子。树洞里匹配到她。 | 场景：空荡荡的房间。 |
| **Ch 26** | Winter | **树洞默契**。不问“你是谁”，只说“我要去一个雨很多的地方”。 | 对话：模糊的信息交换。 |
| **Ch 27** | 嘉弘 | **回应流浪**。回一句“巧了，我也是”。两人以为只是巧合。 | 误会：以为对方在开玩笑。 |
| **Ch 28** | Winter | **机场伪装**。全副武装出现在机场。看着窗外的雨，想起树洞的话。 | 状态：出发。 |

## 🧺 第三阶段：同一座城的孤岛 (The Island)
*关键词：T国雨季、自助洗衣店、物理擦肩、感官互通*

| 章号 | 视角 | 核心事件 | 关键钩子/备注 |
| :--- | :--- | :--- | :--- |
| **Ch 29** | 嘉弘 | **落地北区**。新租屋依然无窗且更潮湿。衣服洗了不干，发霉味。 | **刚需**：必须找洗衣店。 |
| **Ch 30** | Winter | **落地东区**。入住高端公寓/酒店，但严禁外出。窗外全是雨。 | 压抑：新的牢笼。 |
| **Ch 31** | 嘉弘 | **发现据点**。深夜找到巷子里的“24h自助洗衣店”。白噪音让他心安。 | **场景解锁**：洗衣店。 |
| **Ch 32** | Winter | **秘密通道**。发现酒店后门通向一条老街。深夜溜出去透气。 | 动作：第一次叛逆。 |
| **Ch 33** | 嘉弘 | **树洞连线**。在洗衣店里匹配。背景音是同款滚筒声，但他以为是巧合。 | 错位：听觉重叠。 |
| **Ch 34** | Winter | **潜入洗衣店**。第一次走进那家店。角落里坐着一个男生（嘉弘），她没敢靠近。 | **物理相遇**：第一次同框。 |
| **Ch 35** | 嘉弘 | **无声默契**。看到那个包裹严实的女生进来。他把放在椅子上的包拿开，腾出位置。 | 动作：无言的善意。 |
| **Ch 36** | Winter | **树洞吐槽**。在树洞说“今天差点遇到人，幸好他没抬头”。 | 嘉弘在屏幕对面看，没意识到是自己。 |
| **Ch 37** | 嘉弘 | **冷笑话试探**。在树洞讲关于“湿衣服”的冷笑话。Winter 在几米外的椅子上笑出声。 | **高光时刻**：笑声重叠。 |
| **Ch 38** | Winter | **现实观察**。她开始观察这个常在洗衣店的男生。看他打字，看他推眼镜。 | 视角：观察者。 |
| **Ch 39** | 嘉弘 | **职业升级**。在 T 国正式接手 Tier 3 专线。处理更棘手的单子，压力剧增。 | 状态：疲惫。 |
| **Ch 40** | Winter | **练习受挫**。T 国首演排练不顺。想吃冰淇淋，想去洗衣店。 | 渴望：这里是充电站。 |

## ⚡ 第四阶段：危机与隐秘共犯 (The Accomplice)
*关键词：差点掉马、推眼镜动作、共犯结构、第一卷收尾*

| 章号 | 视角 | 核心事件 | 关键钩子/备注 |
| :--- | :--- | :--- | :--- |
| **Ch 41** | 嘉弘 | **加班缺席**。连续两天没去洗衣店。树洞里没上线。 | 状态：缺席。 |
| **Ch 42** | Winter | **空等待**。对着空的 3 号洗衣机发呆。意识到自己习惯了他的存在。 | 心理：依赖产生。 |
| **Ch 43** | 嘉弘 | **回归与道歉**。回到洗衣店，在树洞说“最近工作太忙”。 | 解释。 |
| **Ch 44** | Winter | **面对面树洞**。两人面对面坐着，却在手机里聊天。最安全的距离。 | 场景：极度拉扯。 |
| **Ch 45** | 嘉弘 | **推眼镜**。思考回复时，做了一个标志性的推眼镜动作。 | **掉马前兆**：动作捕捉。 |
| **Ch 46** | Winter | **识别确认**。看到动作，瞳孔地震。确认了：树洞=眼前的男生。 | **Key Moment**：单向掉马。 |
| **Ch 47** | 嘉弘 | **不知情**。继续在树洞发冷笑话。发现对方回复变慢了。 | 悬念：她怎么了？ |
| **Ch 48** | Winter | **试探边界**。在树洞问：“如果我就在你附近呢？” | 试探。 |
| **Ch 49** | 嘉弘 | **理性回答**。回：“那我会装作不认识，让你安全。” | 回答满分，Winter 破防。 |
| **Ch 50** | Winter | **主动靠近**。离开前，在他桌上放了一罐热咖啡（或硬币）。 | 动作：现实互动的突破。 |
| **Ch 51** | 嘉弘 | **看着咖啡**。那是 7-11 的特定款。他开始怀疑，但不敢信。 | 猜疑链启动。 |
| **Ch 52** | Winter | **活动危机**。T 国活动现场出现私生饭冲撞。此时嘉弘正好在处理相关外卖单。 | 外部危机接入。 |
| **Ch 53** | 嘉弘 | **远程介入**。通过富胖达后台发现异常订单（私生饭点的），间接帮了忙。 | 技能高光。 |
| **Ch 54** | Winter | **卷终钩子**。雨季结束的那个晚上。她在树洞问：“雨停了，我要走了吗？” | **Ending**：开放式悬念。 |
